Basic nbody graviational simulation program in c++

Usage
---
To compile:
	g++ nbody.cpp -o nbody.exe -enable-auto-import

To run:
	nbody.exe STEPS DT < INPUTFILE

Example:
	nbody.exe 1000 0.01 < InputFiles/testInput > testOutput.out
